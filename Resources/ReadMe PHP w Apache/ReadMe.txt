

How to use PHP w apache


1) Download Bitnami



2) Go into directory bitnami/ apache2 / htdocs/



3) Copy PHP_Notes directory/folder into bitnami/ apache2 / htdocs/



4) Open Bitnami (manager-osx)



5) Click "Go To Application" button on bottom right corner



6) Open the PHP_Notes directory




7) Go through the PHP files using Apache
